Football Club ERP System

A full-stack Football Club **Enterprise Resource Planning (ERP)** web application built with **React** (frontend) and **Django** (backend).

This application provides a centralized system for managing club operations, including player management, team scheduling, match records, staff accounts, and reporting tools.
